# LMK Linux Builder for Windows

Windows frontend + GitHub Actions Linux build backend for:

- VINABOX X9
- Rockchip RK3229
- TXCZ-RK3229-V2.3
- 2 GB RAM
- Micron 8 GB raw NAND

Run `RUN_LMK_BUILDER.bat`, then follow `HUONG_DAN.txt`.

The generated image is experimental. Do not flash it to raw NAND before
reviewing the loader, partition layout and NAND installation method.
